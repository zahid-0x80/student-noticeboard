# Student Noticeboard

A web application built for Daffodil International University, Department of Software Engineering.

## What is this?

Student Noticeboard is a simple website where teachers can post notices and announcements. Students can view them anytime and get email notifications when a new notice is posted.

## Features

1. Post a notice
2. View all notices
3. Delete a notice
4. Edit a notice
5. Search notices by keyword
6. Filter notices by category
7. Date and time on each notice
8. Mark a notice as important
9. Sort notices by newest or oldest
10. Character counter when typing
11. Total notice count
12. Confirmation popup before deleting
13. No notices yet message
14. Important notices highlighted in red
15. Contact form
16. About Us page
17. Dark mode and light mode toggle
18. Print button for each notice
19. Copy button for each notice
20. AI Chatbox powered by Groq

## Extra Features

- Teacher login and logout system
- Student email notification system
